# Media Recommender — Frontend (React + Vite + Tailwind)

Projeto frontend completo (opção B) - Chat + Perfis + Playlists + UI moderna.

## Como usar localmente

1. Instale dependências:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Abra http://localhost:5173

## Configurar a API

Edite `.env` ou `.env.local` com:

```
VITE_API_BASE_URL=https://seu-projeto.onrender.com
```

## Deploy

Recomendo usar Vercel. Defina a variável `VITE_API_BASE_URL` no painel do Vercel.
